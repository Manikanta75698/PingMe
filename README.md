# 🚀 PingMe - Real Time Chat Application

A modern full-stack real-time messaging application built using React, Node.js, Express, MongoDB and Socket.IO.

## ✨ Features

- 🔐 OTP Authentication
- 💬 Real-time Chat
- 🟢 Online Users
- ✍️ Typing Indicator
- 👀 Message Seen Status
- 📷 Image Sharing
- 👤 Profile System
- ❤️ Follow / Unfollow
- 🌙 Dark Mode
- 📱 Responsive Design

## 🛠 Tech Stack

Frontend:
- React
- CSS
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- Socket.IO
- JWT

Cloud:
- Cloudinary
- Render
- Vercel
