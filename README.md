# 🚀 PingMe - Real Time Chat Application

PingMe is a modern full-stack real-time messaging application that allows users to connect, chat instantly, share images, and manage their profiles with a smooth and premium user experience.

---

## 🌐 Live Demo

🔗 Frontend: https://your-vercel-link.vercel.app

🔗 Backend API: https://pingme-api-new.onrender.com

---

## ✨ Features

### 🔐 Authentication
- User Registration
- Email OTP Verification
- Secure Login
- JWT Authentication
- Protected Routes

### 💬 Real-Time Chat
- Instant Messaging using Socket.IO
- Online Users Status
- Typing Indicator
- Message Seen Status
- Smooth Chat Experience

### 📸 Media Sharing
- Send Image Messages
- Image Preview
- Upload Profile Pictures
- Cloudinary Image Storage

### 👤 Profile System
- View Profile
- Edit Name
- Change Username
- Update Bio
- Follow / Unfollow Users

### 🎨 Modern UI/UX
- Premium Dark Theme
- Instagram Inspired Chat UI
- Modern Chat Bubbles
- Responsive Mobile Design
- Smooth Animations

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/PingMe.git
```

### Install Frontend Dependencies

```bash
cd client
npm install
npm run dev
```

### Install Backend Dependencies

```bash
cd server
npm install
npm start
```
